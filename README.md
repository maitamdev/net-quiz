# 🌐 Ôn Tập Mạng Máy Tính - NetQuiz

Nền tảng ôn tập trắc nghiệm môn **Mạng Máy Tính** dành cho sinh viên. Miễn phí 100%.

## 📚 Nội dung

| Chương | Tên | Số câu |
|--------|-----|--------|
| 1 | Khái niệm cơ bản | 30 |
| 2 | Tầng ứng dụng | 30 |
| 3 | Tầng giao vận | 30 |
| 4 | Tầng mạng | 30 |
| 5 | Tầng liên kết dữ liệu | 30 |
| 6 | Tầng vật lý | 30 |
| 7 | Ôn tập cuối kì | 42 |
| **Tổng** | | **222 câu** |

## ✨ Tính năng

- 📝 **Luyện tập theo chương** - Chọn chương muốn ôn, câu hỏi random mỗi lần
- ⏱️ **Thi thử** - Mô phỏng bài thi 45 phút
- 🔖 **Đánh dấu câu hỏi** - Lưu câu khó để ôn lại
- 💡 **Giải thích chi tiết** - Hiện giải thích ngay sau khi chọn đáp án
- 📊 **Bảng điều khiển** - Theo dõi tiến độ học tập
- 💾 **Lưu tiến trình** - Reload trang vẫn tiếp tục làm bài

## 🛠️ Công nghệ

- **Frontend**: Vite + Vanilla JS
- **Backend**: Supabase (PostgreSQL + Auth)
- **Hosting**: Vercel

## 🚀 Chạy local

```bash
# Clone repo
git clone https://github.com/maitamdev/net-quiz.git
cd net-quiz

# Cài dependencies
npm install

# Tạo file .env.local với Supabase credentials
# VITE_SUPABASE_URL=your_url
# VITE_SUPABASE_ANON_KEY=your_key

# Chạy dev server
npm run dev
```
